Banking

Manage bank accounts and track banking transactions. This module provides tools for recording deposits, withdrawals, and reconciling bank statements.

Who Can Access

Banking is available to Accounts Staff, Managers, and Administrators.

Accessing Banking

Navigate to Banking from the main sidebar menu.

Module Structure

The Banking module has two tabs:

Accounts Tab

Manage your company's bank accounts.

Transactions Tab

Record and track banking transactions.

Accounts Tab

Account List Columns

Column Description
Account Number Bank account number
Account Name Name for the account
Bank Name Name of the bank
Branch Branch name
Account Type Checking or Savings
Current Balance Current account balance
Opening Balance Initial balance when account was added
Status Account status
Actions View, Edit, Delete

Adding a Bank Account

  1. Click Add Account

    Click + Add Account button.

  2. Enter Account Details
    • Account Number *
    • Account Name *
    • Bank Name *
    • Branch Name
    • Branch Code
    • IBAN
    • SWIFT Code
    • Currency - Default: OMR
    • Account Type - Checking or Savings
    • Opening Balance *
    • Notes
  3. Save

    Click Save to create the account.

Transactions Tab

Transaction List Columns

Column Description
Date Transaction date
Type Deposit or Withdrawal
Category Transaction category
Amount Transaction amount
Payee/Payer Who paid or received
Reference Number Reference for the transaction
Reconciliation Status Whether transaction is reconciled
Actions View, Edit, Reconcile, Delete

Recording a Transaction

  1. Click Add Transaction

    Click + Add Transaction button.

  2. Enter Transaction Details
    • Account * - Select bank account
    • Transaction Date *
    • Type * - Deposit or Withdrawal
    • Category *
    • Amount *
    • Reference Number
    • Payee/Payer Name
    • Description
    • Notes
  3. Save

    Click Save to record the transaction. The account balance updates automatically.

Filtering Transactions

Use the following filters to find specific transactions:

  • Account - Filter by bank account
  • Transaction Type - Deposit or Withdrawal
  • Category
  • Reconciliation Status
  • Date Range
  • Search - Search by description or reference

Reconciling Transactions

Mark transactions as reconciled when they match your bank statement:

  1. Find the transaction in the list
  2. Click the Reconcile button
  3. Confirm the reconciliation

Reconciled transactions are marked and help ensure your records match the bank's records.
