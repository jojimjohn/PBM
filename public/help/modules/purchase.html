Purchase Module

The Purchase module is a central hub for managing material procurement. It includes collections, purchase orders, vendor billing, and purchase-related expenses.

Who Can Access

Purchase Staff, Managers, and Administrators.

Accessing Purchase

Navigate to Purchase from the main sidebar menu. The module has four tabs.

Module Structure (4 Tabs)

The Purchase module is organized into four tabs:

1. Collections

Manage the materials collection workflow including callouts, collection orders, and WCN finalization.

2. Orders

View and manage purchase orders - both auto-generated from collections and manually created.

3. Bills

Track company bills and vendor bills. Supports multi-PO vendor billing.

4. Expenses

Record and track purchase-related expenses like transportation, labor, etc.

Tab 1: Collections

The Collections tab manages the process of collecting materials from suppliers.

Collections Workflow

  1. Create Callout

    Schedule a collection visit to a supplier location. Specify supplier, location, date/time, and expected materials.

  2. Create Collection Order

    When collection occurs, create a collection order to document actual materials collected and quantities.

  3. Finalize WCN

    Finalize the Waste Consignment Note (WCN). This automatically:

    • Updates inventory with collected materials
    • Splits composite materials into components
    • Generates a purchase order with contract rates
  4. WCN Rectification (if needed)

    If errors are found after finalization, use WCN Rectification to make corrections.

Collection Workflow Stages

Collections are managed through the Workflow Progress Bar with four stages:

  • Scheduled (Yellow) - Collection created, awaiting driver assignment or dispatch
  • In Progress (Blue) - In Transit or Collecting - driver is active
  • Completed (Green) - Collection finished, ready for WCN finalization
  • Finalized (Indigo) - WCN complete, PO auto-generated

Click any stage in the Workflow Progress Bar to filter the table.

Tab 2: Orders

View and manage purchase orders.

Order List Columns

Column Description
Order Number Unique PO identifier
Supplier Name Supplier for this order
Order Date Date order was created
Items Count Number of line items
Total Amount Order total (OMR)
Status Draft, Pending, Approved, Sent, Received, Completed, Cancelled
Amendment Count Number of amendments (badge)
Actions View, Edit, Amend, Receive, Delete

Creating a Manual Purchase Order

  1. Click + New PO
  2. Select supplier
  3. Add line items with material, quantity, and price
  4. Set payment terms and notes
  5. Attach documents if needed
  6. Submit for approval

PO Amendments

To modify an existing PO:

  1. Open the PO
  2. Click Create Amendment
  3. Make changes
  4. Submit for approval

All amendments are tracked for audit purposes.

Tab 3: Bills

Track vendor invoices and payment status.

Bill Types

Company Bills

Bills generated by your company. One PO equals one bill.

Vendor Bills

Invoices received from vendors. One vendor invoice can cover multiple POs.

Bill Filters

  • Bill Type - All, Company, Vendor
  • Payment Status - All, Unpaid, Partial, Paid, Overdue

Bill List Columns

Column Description
Invoice Number Bill/Invoice identifier
Supplier Name Vendor name
Bill Date Invoice date
Invoice Amount Total amount
Paid Amount Amount paid so far
Balance Due Remaining balance
Payment Status Unpaid, Partial, Paid, Overdue
Associated POs Number of linked POs
Actions View, Pay, Edit, Delete

Tab 4: Expenses

Track purchase-related expenses such as transportation, labor, and other costs.

Expense List Columns

Column Description
Reference PO number or other reference
Category Expense category
Description Expense description
Amount Expense amount (OMR)
Date Expense date
Vendor Vendor name
Status Recorded, Pending, Approved, Rejected
Actions View, Edit, Delete

Recording an Expense

  1. Click + Add Expense
  2. Enter details:
    • PO Reference - Link to purchase order
    • Category
    • Description
    • Amount
    • Date
    • Vendor Name
  3. Attach receipt if available
  4. Click Save
