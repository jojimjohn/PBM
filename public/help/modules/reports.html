Reports

Generate and analyze business reports. The Reports module provides insights into purchases, collections, expenses, and vendor billing.

Who Can Access

Managers and Administrators have full report access. Other roles may have limited access based on their function.

Accessing Reports

Navigate to Reports from the main sidebar menu.

Available Reports

The system currently provides four reports:

1. Purchase Cost Analysis

Analyze purchase costs by supplier and period.

  • Supplier breakdown
  • PO Number reference
  • Date tracking
  • Base cost vs actual cost
  • Collection expenses included

2. WCN Register

Track all Waste Consignment Notes.

  • WCN Number
  • Date
  • Supplier
  • Material details
  • Quantity
  • Status (Pending/Finalized/Rectified)

3. Collection Expenses

Review all expenses related to material collection.

  • Category (Transportation/Labor/Other)
  • Description
  • Amount
  • Date
  • Approval status
  • Expense breakdown charts

4. Vendor Bill Tracking

Track vendor invoices and payment status.

  • Invoice number
  • Supplier
  • Date
  • Invoice amount
  • Paid amount
  • Balance due
  • Payment status

Generating Reports

  1. Select Report Type

    Choose from the available reports: Purchase Cost Analysis, WCN Register, Collection Expenses, or Vendor Bill Tracking.

  2. Set Date Range

    Use the date range filter to specify the period. Preset options include:

    • Today
    • This Week
    • This Month
    • This Quarter
    • This Year
    • Custom Range
  3. Apply Additional Filters

    Each report has specific filters:

    • Purchase Cost Analysis: Supplier filter
    • WCN Register: Status filter (Pending/Finalized/Rectified)
    • Collection Expenses: Category filter (Transportation/Labor/Other)
    • Vendor Bill Tracking: Supplier, Bill Type (Company/Vendor), Payment Status (Unpaid/Partial/Paid/Overdue)
  4. View Results

    The report displays with:

    • Data table with all matching records
    • Summary statistics
    • Charts where applicable
  5. Export (Optional)

    Export the report data:

    • CSV: For spreadsheet analysis
    • PDF: For printing and sharing

Report Details

Purchase Cost Analysis

This report shows the breakdown of purchase costs including collection expenses:

Column Description
Supplier Name of the supplier
PO Number Purchase order reference
Date Transaction date
Base Cost Material purchase cost
Collection Expenses Associated collection costs
Actual Cost Total cost (base + expenses)

Includes cost trend charts over time.

WCN Register

Complete listing of all Waste Consignment Notes:

Column Description
WCN Number Unique WCN identifier
Date Collection date
Supplier Supplier name
Material Material collected
Quantity Amount collected
Status Pending, Finalized, or Rectified

Collection Expenses

Detailed breakdown of collection-related expenses:

Column Description
Category Transportation, Labor, or Other
Description Expense details
Amount Expense amount (OMR)
Date Expense date
Approval Status Recorded, Pending, Approved, Rejected

Includes pie chart for expense breakdown by category and trend line chart.

Vendor Bill Tracking

Monitor vendor invoices and payment status:

Column Description
Invoice # Vendor invoice number
Supplier Vendor/Supplier name
Date Invoice date
Amount Total invoice amount
Paid Amount paid to date
Balance Remaining balance due
Status Unpaid, Partial, Paid, or Overdue

Summary shows: Total Invoiced, Total Paid, Total Due.
