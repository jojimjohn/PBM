Supplier Management

Manage your supplier relationships and their collection locations. Suppliers are essential for the collections workflow and purchase order processing.

Who Can Access

Supplier Management is available to Purchase Staff, Managers, and Administrators.

Accessing Suppliers

Navigate to Suppliers from the main sidebar menu. This is a top-level navigation item appearing after Customers.

Supplier List

The supplier list displays all suppliers with the following columns:

Column Description
Supplier Supplier name, code, and type
Contact Contact person and phone number
Specialization/Region Business specialization and location
Rating Supplier rating
Payment Terms Payment terms (days)
Status Active or Inactive badge
Actions View, Edit, Location Manager, Delete

Adding a New Supplier

  1. Click Add Supplier

    Click + Add Supplier in the page header.

  2. Enter Company Details
    • Supplier Code - Auto-generated
    • Name * - Official business name
    • Type - Business or Individual
    • Contact Person * - Primary contact
    • Phone * - Contact number
    • Email - Contact email
    • VAT Registration - VAT number if applicable
    • Address - Business address
    • City - City name
    • Region - Select from dropdown
  3. Configure Payment and Specialization
    • Payment Terms - Days for payment (e.g., 30 days)
    • Specialization - Multi-select checkboxes for material types
  4. Enter Banking Details
    • Bank Name
    • Account Number
    • IBAN
    • Tax Number
  5. Add Notes and Save

    Add any notes and click Save.

Viewing Supplier Details

Click the View button to open the supplier details modal with the following sections:

Overview

  • Name and code
  • Type
  • Contact person
  • Phone and email

Address

  • Street address
  • City
  • Region

Banking

  • Bank name
  • Account number
  • IBAN
  • Tax number

Terms

  • Payment terms
  • Specialization

Managing Supplier Locations

Each supplier can have multiple collection locations. Access the Location Manager by clicking the Locations button in the actions column.

Adding a Location

  1. Click Locations button for the supplier
  2. Click + Add Location
  3. Enter location details:
    • Location Name - Descriptive name (e.g., "Main Factory")
    • Location Code - Short code for reference
    • Address - Full physical address
    • Contact - Site-specific contact person
    • Instructions - Special notes for collectors/drivers
  4. Click Save
Locations and Contracts

Supplier locations are used in contracts to define location-specific material rates. Add all locations before creating contracts.
