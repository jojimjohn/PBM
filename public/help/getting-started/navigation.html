System Navigation

Master the PBM interface to work efficiently. Learn about the header navigation, and quick navigation techniques to access any feature instantly.

Main Navigation

The main navigation is located in the header bar, below the company logo and user controls. It organizes all system modules into logical groups for easy access.

Navigation Structure

Section Contents Access
Dashboard Main dashboard with KPIs and charts All Users
Sales Customers, Contracts, Sales Orders, Invoices Sales Admin
Purchase Suppliers, Purchase Orders, Collections Purchase Admin
Inventory Materials, Stock Levels, Wastage All Users
Finance Petty Cash, Expenses, Transactions Accounts Admin
Reports Analytics, Export, Audit Logs Manager Admin
Settings Users, Roles, System Configuration Admin
Role-Based Visibility

You'll only see menu items you have permission to access. If you need access to a hidden section, contact your administrator.

Header Bar

The header bar at the top of every page provides essential controls and information:

Company Indicator

Shows which company you're currently logged into (Al Ramrami or Pride Muscat). The logo and color scheme reflect the active company.

Search Bar

Quick search across all accessible data. Type customer names, order numbers, or keywords to find information fast.

Notifications

Bell icon showing pending alerts. Red badge indicates unread notifications requiring your attention.

User Menu

Your profile dropdown with options for Settings, Help, and Logout.

Page Structure

Most pages in PBM follow a consistent layout for easy learning:

  1. Page Header

    Contains the page title, action buttons (like "Add New"), and filtering options.

  2. Filter Bar

    Search boxes, date ranges, and dropdown filters to narrow down displayed data.

  3. Data Table

    The main content area displaying records in rows. Click column headers to sort. Click rows to view details.

  4. Pagination

    Navigate between pages of data. Choose how many items to display per page.

Common Actions

Learn these standard actions that work throughout the system:

Action How To Where It Works
Create New Click the "Add" or "+" button in the page header All list pages
View Details Click on a row in the data table, or click the "View" icon All list pages
Edit Click the pencil/edit icon in the row actions Most editable records
Delete Click the trash icon (confirmation required) Records you have permission to delete
Filter Use the search box or dropdown filters above the table All list pages
Sort Click a column header to sort ascending/descending All data tables
Export Click the "Export" button to download data as CSV/Excel Most list pages

Working with Forms & Modals

Data entry happens in modal dialogs (popup forms):

Form Basics

  • Required Fields: Marked with a red asterisk (*)
  • Validation: Errors appear below fields in red text
  • Save: Click "Save" or "Submit" to confirm changes
  • Cancel: Click "Cancel" or the X to close without saving
  • Keyboard: Press Enter to submit, Esc to cancel
Unsaved Changes

If you close a form with unsaved changes, you may see a confirmation dialog. Make sure to save important work before navigating away.

Tabs and Sections

Complex pages use tabs to organize related information:

Example: Petty Cash Page

  • Cards Tab: Manage petty cash cards
  • Expenses Tab: View, submit, and approve expenses
  • Users Tab: Manage card holders

Click a tab to switch views. The URL updates to reflect your current tab, so you can bookmark specific sections.

Keyboard Shortcuts

Power users can navigate faster with keyboard shortcuts:

Shortcut Action
/ Focus the search bar
Esc Close modal dialog
Enter Submit form
Tab Move to next form field
Shift + Tab Move to previous form field

Language Support

PBM supports both English and Arabic interfaces:

Switching Languages

  1. Click your profile icon in the header
  2. Select "Settings" from the dropdown
  3. Find the "Language" option
  4. Choose between English and Arabic (العربية)
  5. The interface will update immediately
RTL Support

When Arabic is selected, the entire interface switches to Right-to-Left (RTL) layout for natural reading direction.
