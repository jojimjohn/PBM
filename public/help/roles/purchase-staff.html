Purchase Staff Guide

As Purchase Staff, you manage supplier relationships, coordinate collections, and handle purchase orders. This guide covers the collections workflow and procurement processes.

Purchase Staff Role

Your System Access

Full Access

  • Supplier Management
  • Supplier Contracts
  • Callouts
  • Collection Orders
  • Purchase Orders
  • Wastage Records

View Only

  • Inventory levels
  • Material information
  • Your purchase reports

No Access

  • Customer management
  • Sales orders
  • System settings
  • User management

Collections Workflow

The primary workflow for contract-based purchases:

  1. Create Callout

    Schedule a collection visit to a supplier location:

    • Select supplier and location
    • Set scheduled date/time
    • Note expected materials
  2. Execute Collection

    Record the actual collection:

    • Create Collection Order from callout
    • Enter actual materials and quantities
    • Record any variances from expected
  3. Finalize WCN

    Finalize the Waste Consignment Note:

    • Review collection details
    • Click Finalize
    • System updates inventory automatically
    • Auto-generates purchase order
  4. Process PO

    The auto-generated PO contains contract rates. Review and process for payment.

Key Tasks

Managing Suppliers

  • Add new suppliers with all collection locations
  • Keep contact information updated
  • Configure all location addresses and instructions
  • Monitor supplier contract terms

Recording Wastage

When material loss occurs:

  1. Go to Inventory → Wastage
  2. Click + Record Wastage
  3. Select material and enter quantity lost
  4. Choose wastage type (spillage, damage, etc.)
  5. Provide detailed description
  6. Submit for manager approval
Record Immediately

Report wastage as soon as it happens. Delayed reporting raises audit concerns.

Manual Purchase Orders

For non-contract purchases:

  1. Go to Purchase → Purchase Orders
  2. Click + New PO
  3. Select supplier
  4. Add materials with quantities and prices
  5. Set payment terms
  6. Submit for approval

Purchase Best Practices

  • Plan collections: Schedule based on inventory needs and supplier availability
  • Verify quantities: Always count/measure materials during collection
  • Document issues: Note any problems during collection immediately
  • Finalize promptly: Don't delay WCN finalization - it blocks inventory updates
  • Track supplier performance: Note reliability and quality issues
  • Monitor contracts: Alert management before supplier contracts expire

Purchase Staff Daily Checklist

  • ☐ Review today's scheduled callouts
  • ☐ Check pending collection orders to finalize
  • ☐ Review purchase orders awaiting approval
  • ☐ Check low stock items needing collection
  • ☐ Follow up on pending supplier issues
  • ☐ Record any wastage that occurred
