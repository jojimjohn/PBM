Administrator Guide

As a System Administrator, you have full access to all system features including user management, system configuration, and security settings. This guide covers your key responsibilities.

Administrator Role

Your Responsibilities

User Management

  • Create and manage user accounts
  • Assign roles and permissions
  • Reset passwords
  • Deactivate users

System Configuration

  • Configure session timeout
  • Set up system preferences
  • Manage materials and categories
  • Configure approval workflows

Data Management

  • Create/edit customers and suppliers
  • Manage contracts
  • Configure petty cash cards
  • System backups

Security & Audit

  • Monitor system access
  • Review audit logs
  • Investigate security events
  • Ensure compliance

User Management

Creating a New User

  1. Navigate to Users

    Go to Settings → Users.

  2. Click Add User

    Click + Add User button.

  3. Enter User Details
    • Full Name *
    • Email * - Used for login
    • Initial Password * - User should change on first login
    • Role * - Select appropriate role
  4. Assign Role

    Choose the appropriate role:

    • Admin: Full system access
    • Manager: Approval authority + reports
    • Sales Staff: Sales and customer operations
    • Purchase Staff: Purchase and supplier operations
    • Accounts Staff: Financial operations
  5. Save

    Click Save. The user can now log in.

Security Reminder

Always use strong initial passwords and require users to change them on first login. Follow the principle of least privilege when assigning roles.

System Settings

Session Timeout Configuration

Configure how long users can be inactive before being logged out:

  1. Go to Settings → System
  2. Find Session Timeout
  3. Set the timeout period (default: 30 minutes)
  4. Save changes

Petty Cash Settings

  • Approval Limits: Set amounts requiring manager approval
  • Categories: Configure expense categories
  • Receipt Requirements: Enable mandatory receipt uploads

Petty Cash User Management

As admin, you manage PC Users who access the mobile expense portal:

  1. Create PC User

    Go to Petty Cash → Users and add new PC user with name, phone, and assigned card.

  2. Generate Credentials

    System creates QR code and PIN automatically.

  3. Share Securely

    Provide QR and PIN to user through secure channel (not email).

  4. Monitor Activity

    View user's expense submissions and deactivate if needed.

Audit Logs

Monitor system activity through audit logs:

What's Logged

  • User logins and logouts
  • Failed login attempts
  • Data creation, updates, and deletions
  • Approval actions
  • System configuration changes
  • Session timeouts

Reviewing Logs

  1. Go to Settings → Audit Logs
  2. Filter by date range, user, or action type
  3. Export logs for compliance reporting

System Backups

PBM includes automated backup capabilities:

  • Automatic Backups: Daily backups at configured time
  • Manual Backups: Trigger on-demand backup before major changes
  • Backup Retention: Configure how long to keep old backups
Best Practice

Always create a manual backup before making major system configuration changes or data imports.

Admin Daily Checklist

  • ☐ Review any failed login attempts in audit logs
  • ☐ Check for pending user requests or issues
  • ☐ Verify backup completed successfully
  • ☐ Review system performance and errors
  • ☐ Check expiring contracts and user accounts
