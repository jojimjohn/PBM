Accounts Staff Guide

As Accounts Staff, you manage petty cash, track expenses, and handle financial operations. This guide covers petty cash management and expense tracking workflows.

Accounts Staff Role

Your System Access

Full Access

  • Petty Cash Cards
  • PC Users
  • Expense Management
  • Financial Reports

View Only

  • Inventory reports
  • Sales summaries
  • Purchase summaries

No Access

  • Customer management
  • Supplier management
  • System settings
  • User management

Petty Cash Management

Your primary responsibility is managing the petty cash system.

Managing Cards

  1. Create Cards

    Set up petty cash cards for departments:

    • Assign descriptive names
    • Set initial balance
    • Configure monthly limits if needed
  2. Monitor Balances

    Regularly check card balances and reload before they run out.

  3. Reload Cards

    Add funds to cards when balances are low. Document reload amounts.

Managing PC Users

  1. Add PC Users who will submit expenses via mobile portal
  2. Generate QR code and PIN for each user
  3. Distribute credentials securely
  4. Monitor user activity and deactivate if needed

Processing Expenses

Submitting Expenses (Admin Entry)

You can enter expenses on behalf of others:

  1. Go to Petty Cash → Expenses
  2. Click + Add Expense
  3. Select the appropriate card
  4. Enter amount, category, and description
  5. Optionally assign to a PC user
  6. Attach receipt if available
  7. Submit for approval

Reviewing Submitted Expenses

  • Monitor expenses submitted by PC Users
  • Verify receipts are attached and legible
  • Check categorization is correct
  • Flag issues to manager for rejection

Card Reconciliation

Regularly reconcile petty cash cards:

Monthly Reconciliation Process

  1. Generate card statement for the month
  2. Compare system balance to physical cash
  3. Identify any discrepancies
  4. Document and explain variances
  5. Report issues to manager
Best Practice

Reconcile cards at least monthly, ideally weekly for high-activity cards.

Key Reports

Daily

  • Pending expense approvals
  • Today's expense submissions
  • Low balance alerts

Weekly

  • Expense summary by card
  • Expense by category
  • Card balance report

Monthly

  • Full petty cash report
  • Reconciliation summary
  • Expense trends

Best Practices

  • Enforce receipts: Ensure all expenses have receipts attached
  • Check categorization: Verify expenses are properly categorized
  • Monitor limits: Watch for cards approaching monthly limits
  • Timely reloads: Reload cards before they run out
  • Regular reconciliation: Don't skip monthly reconciliation
  • Document issues: Keep records of any discrepancies

Accounts Staff Daily Checklist

  • ☐ Check pending expense submissions
  • ☐ Review card balances for low balance alerts
  • ☐ Verify receipts on recent submissions
  • ☐ Follow up on rejected expenses
  • ☐ Process any card reload requests
  • ☐ Review PC user activity
