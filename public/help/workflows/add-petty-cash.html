Adding Petty Cash Cards

This guide walks you through creating and configuring petty cash cards, setting up PC Users, and managing card operations.

Required Role

Accounts Staff or Administrator

Creating a New Petty Cash Card

  1. Navigate to Petty Cash

    Go to Finance â†’ Petty Cash and ensure you're on the Cards tab.

  2. Click Add Card

    Click the + Add Card button in the top-right corner.

  3. Enter Card Details

    Fill in the required information:

    • Card Name * - Descriptive name (e.g., "Operations Petty Cash", "Sales Team Card")
    • Card Number * - Unique identifier (e.g., "PC-001")
    • Department - Which department uses this card
    • Initial Balance * - Starting amount on the card
    • Monthly Limit - Maximum spending per month (optional)
    Naming Convention

    Use clear, descriptive names that identify the card's purpose. This helps with reporting and reconciliation.

  4. Add Notes (Optional)

    Add any notes about how this card should be used or any restrictions.

  5. Save the Card

    Click Save. The card is now active and ready for use.

Adding PC Users to a Card

PC Users can submit expenses via the mobile portal.

  1. Go to Users Tab

    In Petty Cash, click on the Users tab.

  2. Click Add PC User

    Click + Add PC User.

  3. Enter User Details
    • Name * - Full name of the user
    • Email - Contact email
    • Phone * - Mobile number
    • Assigned Card * - Select the petty cash card they'll use
  4. Save and Get Credentials

    Click Save. The system generates:

    • QR Code - For portal access
    • 4-Digit PIN - For authentication
  5. Share Credentials Securely

    Provide the QR code and PIN to the user through a secure method:

    • In person (recommended)
    • Encrypted message
    • NOT via regular email
    Security

    Remind users to keep their PIN secret and report if compromised.

Reloading a Card

When a card's balance runs low:

  1. Open Card Details

    Click on the card you want to reload in the Cards tab.

  2. Click Reload Card

    Click the Reload Card button.

  3. Enter Amount

    Enter the amount to add to the card balance.

  4. Add Note

    Add a note explaining the reload (e.g., "Monthly replenishment").

  5. Confirm

    Click Confirm. The balance is updated immediately.

Other Card Operations

Suspending a Card

Temporarily disable a card:

  1. Open card details
  2. Click Suspend Card
  3. Card becomes inactive
  4. Can reactivate later

Viewing History

See all card transactions:

  1. Open card details
  2. Go to History tab
  3. View all expenses and reloads
  4. Export if needed

Closing a Card

Permanently deactivate:

  1. Open card details
  2. Click Close Card
  3. Confirm the action
  4. Card is archived (not deleted)

Best Practices

  • Use meaningful names: Card names should clearly identify purpose
  • Set appropriate limits: Monthly limits prevent overspending
  • Regular monitoring: Check card balances weekly
  • Timely reloads: Don't let cards run empty
  • Secure credentials: Share QR/PIN securely, not via email
  • Monthly reconciliation: Compare system to physical cash
